# Student Management

## Description

Student Management is an Odoo module designed to manage and track student information and their final projects (PFE - Projet de Fin d'Études). 

This addon provides a comprehensive solution for:
- Student records and information management
- Final Project (PFE) tracking and organization
- Role-based access control and security
- Reports for student and project data

## Features

- **Student Module**: Create and manage student profiles with relevant information
- **PFE Module**: Track final projects, deadlines, and project details
- **Security**: Role-based access control with defined user permissions
- **Reports**: Generate detailed reports for students and projects

## Installation

1. Copy this module to your Odoo addons directory
2. Restart the Odoo server
3. Go to Apps and search for "Student Management"
4. Click Install

## Dependencies

This module requires a standard Odoo installation with the base module.
